If a thing is worth doing

By Nathan Cheever

There’s the old saying, “If a thing is worth doing, it’s worth doing well.” I, like many of you, was raised on this saying and others like it. “If you’re going to do a job, do it right the first time.” That was another of my mom’s favorites. These sayings are aiming at doing good work rather than sloppy careless efforts.

If I’m going to clean the kitchen, do a good job wiping off the counters, getting the toast crumbs under the toaster, and cleaning out the sink. This sentiment is embraced in many more antithetical aphorisms like, “No pain, no gain,” or “Go big or go home.”

Reading the following quote from Mr. G.K. Chesterton made me think again:

“If a thing is worth doing, it’s worth doing badly.”

What about undertaking starting a business? If I think “well, if it’s worth doing, it’s worth doing well.” On a certain level, that means it’s only worth doing if you can do it well. And if you can’t do it well, if you’re not a professional, if you don’t ahve the skills, then it’s not worth doing. And this is a vicious lie.

There are many undertakings for which I am unskilled and unqualified. I will not do them well at all, especially at the start. But should I not do them? Is it not worth doing if I can’t do it well? If I can’t do something well, is it not worth doing at all?

The difference I think is one of effort versus intent.
